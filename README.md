# CI/CD Deployment Dashboard

A modern React.js application for managing CI/CD deployments with an e-commerce-inspired UI design.

## Features

- **Dashboard Overview** - Real-time metrics and deployment status
- **Deployment Management** - E-commerce-style catalog for deployments
- **Pipeline Visualization** - Monitor CI/CD pipeline stages
- **Environment Management** - Track multiple deployment environments
- **Analytics & Insights** - Performance metrics and trends

## Tech Stack

- React 18
- React Router
- Framer Motion
- Recharts
- Lucide React
- CSS3

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Mosrichard/Reactapp-Deployment-Using-Actions.git

# Navigate to project directory
cd Reactapp-Deployment-Using-Actions

# Install dependencies
npm install

# Start development server
npm start
```

The application will open at `http://localhost:3000`

### Available Scripts

- `npm start` - Runs the app in development mode
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run eject` - Ejects from Create React App
